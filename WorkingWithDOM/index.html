Working With the DOM

We will look at the different DOM methods we can use to select elements from the page. We need to select elements before we can work with them programmatically.

Elements are represented by objects which contains useful properties and methods for working with them.

  • Properties and methods for working with elements:
    • id property
    • className or classList
    • innerHTML / outerHTML or textContent
    • setAttribute
    • removeAttribute
    • hasAttribute

DOM Traversal - Traversing the DOM from one element to another

  • Several methods for moving down the DOM:
  • childNodes – all node types
  • children – only element nodes
  • Using parentNode to move up the DOM
  • previousSibling / nextSibling to move left / right
